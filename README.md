# Project Name: Hotel Booking App

## Description
Мобильное приложение для бронирования отелей с возможностью выбора транспорта (поезд, автобус, самолёт), построения маршрута на карте и просмотра списка рейсов с ценами. Приложение поддерживает регистрацию пользователей и локальное хранение данных в SQLite. Разработано на Swift с использованием SwiftUI.

## Installation
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/PolinaKalchevskaya/mobile-app-hotel-booking.git
   ```
2. Откройте проект в Xcode:
   ```bash
   cd [название-репозитория]
   open HotelBooking.xcodeproj
   ```
3. Выберите целевое устройство (симулятор iPhone или реальный iPhone с iOS 15+).
4. Нажмите `Cmd + R` или кнопку **Run** для сборки и запуска приложения.

## Usage
После успешной установки и запуска приложения:

1. **Регистрация / Вход**: Создайте новый аккаунт или войдите в существующий.
2. **Выбор отеля**: Просмотрите список доступных отелей, выберите подходящий вариант.
3. **Бронирование**: Укажите даты заезда и выезда, подтвердите бронирование.
4. **Выбор транспорта**: Выберите вид транспорта (поезд / автобус / самолёт).
5. **Маршрут на карте**: Постройте маршрут до отеля с помощью встроенной карты.
6. **Список рейсов**: Просмотрите доступные рейсы с названием компании и стоимостью, подтвердите поездку.
